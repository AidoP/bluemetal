# Bluemetal
A toy operating system for RISC-V.

## Dependencies
- [Nightly Rust Toolchain](https://rustup.rs/)
- `clang`
- `qemu-system-riscv64`

## Configuration
See the `configuration` crate.

Set the target hardware with the `MACHINE` environment variable.

## Building
`cargo build`

## Running
`cargo run`
